# Fixed query list for the search parity gate.

# Lidarr lowercases and trims what a user types before sending it, so these
# are written the way they arrive at the server. Each line is one artist
# search. Comments and blank lines are ignored.
#
# The list is deliberately awkward. Parity on "radiohead" proves nothing;
# parity on a band whose name is an article, a punctuation mark, or a
# substring of a thousand other names is the actual test.
